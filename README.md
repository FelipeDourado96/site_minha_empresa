###### Melhorias do projeto:

Aqui se encontram as melhorias do projeto

### Lista de compras

* Trigo
* Fermento 
* Cenoura
* Batata


### Passos de preparo
1. Misture os ovos com o leite, o trigo e o fermento
2. Leve a mistura ao forno a 180º por 20 minutos
3. Retire do forno e desenforme
4. Sirva
